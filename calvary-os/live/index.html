In development
Live — performance tools for worship
The Live module is where Calvary OS becomes a tool you can actually play through. Pads, click tracks, cues and loop triggering. Built to sit alongside the band on Sunday, not just before it.

What's coming

Three phases, shipped one at a time.

Next
Pad player
Play a warm ambient pad in any key, beneath the worship set. Volume, crossfade between keys, seamless loop. Works on laptop, tablet and phone.
After that
Click & cues
In-ear metronome synced to the pad. Section markers so the band knows when the bridge is coming. A rehearsal companion that runs Sunday too.
Later
Loop triggering
Launch pre-recorded loops (bass, drums, vocal ad-libs) with a tap. Extend a chorus on the fly. Build fuller arrangements live.

Where things stand

Honest status — this section exists so you can see the Live module in the nav, not because it does anything yet.

When will the pad player be ready?
Once the pad library is acquired and uploaded. Bolaji is working on sourcing Producer Pads from MultiTracks. Build will follow from there.
Who can use the Live module?
Anyone with worship-team access (same access rule as Rehearsal Studio). Admins can see everything.
What should I use in the meantime?
For rehearsing arrangements and learning parts, use Rehearsal Studio →. For Sunday pads, existing worship pad tools (Coresound Pads Live app, Sunday Keys, etc.) still work.

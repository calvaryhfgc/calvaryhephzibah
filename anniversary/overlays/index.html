Lower & upper third overlays.

Sixteen broadcast graphics for the 34th anniversary service. Each is a 1920×1080 PNG with a transparent zone — drop it into OBS as a full-screen image source above the camera scene and only the obsidian strip renders. Previewed here over a real worship-team frame so you can judge how each one sits.

How to use these in OBS

One-time setup, fast switching

  1. Download the PNG files from this page using the PNG button on each card.
  2. In OBS, create a new scene Service · Overlays with the camera feed as the base source.
  3. Add one Image source per overlay PNG, set each to 1920×1080, locked. Keep them all hidden by default.
  4. During service, toggle visibility on the overlay you need. Show for 5–8 seconds (name supers, song titles) or 10–15 seconds (anniversary banner) and hide.
  5. Never run two overlays at once — they share the same screen zone.

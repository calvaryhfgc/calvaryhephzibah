# Calvary Anniversary — Anniversary Site UI Kit

A high-fidelity recreation of the existing **THANKFUL 2026** anniversary microsite (https://calvaryhfgc.github.io/calvaryhephzibah/calvary-os/anniversary-planning.html), rebuilt against the Calvary design system tokens.

> ⚠️ Recreated from rendered text + structure (no codebase access). Visual treatment follows the Calvary Anniversary visual foundations: Obsidian + Calvary Gold, Anton display, Instrument Serif accents, Inter Tight body, cinematic spotlight backdrops.

## Files

| File | Component |
|---|---|
| `index.html` | Full microsite — interactive scroll prototype |
| `Nav.jsx` | Sticky top nav with backdrop blur |
| `Hero.jsx` | "THANKFUL" mega-word countdown hero |
| `VisionSection.jsx` | "More Than Celebration" + four pillar cards |
| `TimelineSection.jsx` | Critical-path countdown with date chips |
| `CTASection.jsx` | Closing call to action |
| `Footer.jsx` | Word-mark + scripture footer |

## Run
Open `index.html` directly. React + Babel are loaded from CDN.
